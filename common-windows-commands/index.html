# 截屏：快捷键是 Print Screen

  • Ctrl + PrScrn 使用这个组合键截屏，获得整个屏幕的图片。
  • Alt + PrScrn 这个组合键截屏，获得当前窗口的图片。

按下拷屏键之后，系统看似没有任何反应，但系统早已在你按下该键的同时把整个屏幕的内容以图形的形式保存在内存中了，你只要打开一个图像编辑软件（如 Windows 自带的 “画图” 软件、Office 或 photoshop），按下键盘快捷键 “Ctrl+V” 将画面粘贴下来，最后保存为图形文件就可以了。

  • win10 平台下：win + Shift + s 可进行截屏。

# Windows + R：打开 “运行” 窗口（在输入栏输入相关指令则作出相应的操作）

  1. calc - - - - 启动计算器（程序员必备）
  2. notepad - - - - 打开记事本（必备之二）
  3. mspaint - - - - 画图板（截屏完可以用该指令调出画板，把截图粘贴出来）
  4. write - - - - 写字板（类似 word 文档，但功能没那么强大）
  5. taskmgr - - - - 任务管理器（利用快捷键 Ctrl + Alt + Delete 或者 Ctrl + shift + Esc 也可以打开，前者还要再选择启动任务管理器，后者直接启动）
  6. cmd - - - - 进入 CMD 命令提示符窗口

# 任务管理器无法强制结束进程解决方法

  • 第一种：利用进程名结束进程
  1. 进入命令提示符窗口，
  2. 在命令提示符窗口中输入：tasklist
  3. 按回车键，显示出当前运行的程序
  4. 输入 “ Taskkill/im 映像名称 /f ” （eg：Taskkill/im QQ.exe /f）
  5. 按回车键，就会提示 “成功：已中止进程”
  • 第二种：利用 PID 号结束进程
  1. 同样进入命令提示符窗口，
  2. 在命令提示符窗口中输入：tskill PID 号 （PID 就是各进程的身份标识，程序一运行系统就会自动分配给进程一个独一无二的 PID）PID 号可以从任务管理器那里查看，也可以像上面那样输入 tasklist 显示出来；在任务管理器中查看进程时，总是在不断变换位置，很容易看错行，这个时候只要按住 Ctrl 键，就能让 Windows 动态显示瞬间暂停，定格在按下键盘时的那个瞬间，此招适用于 Windows XP 及以上版本的所有系统。win8 好像不能用 tskill 命令
  3. 最后按回车键

# 命令行打开回收站

  • 进 cmd 窗口，输入 start shell:recyclebinfolder 命令可以打开回收站；

或者

  • win + r 打开 “运行”，输入 shell:recyclebinfolder 打开回收站。

# 常用 Windows 组合键

  1. Windows 键：打开开始屏幕
  2. Windows 键 + 空格键：切换输入语言和键盘布局
  3. Windows 键 +,：临时查看桌面
  4. Windows 键 + C：打开 Charms 栏 (提供设置、设备、共享和搜索等选项)
  5. Windows 键 + I：打开设置栏
  6. Windows 键 + Q：打开应用搜索面板
  7. Windows 键 + Tab：循环切换应用
  8. Windows 键 + Shift+Tab：反向循环切换应用
  9. Windows 键 + X：快捷菜单
  10. Windows 键 + ←：最大化窗口到左侧的屏幕上 (与 Metro 应用无关)
  11. Windows 键 + →：最大化窗口到右侧的屏幕上 (与 Metro 应用无关)
  12. Windows 键 + ↑：最大化窗口 (与 Metro 应用无关)
  13. Windows 键 + ↓：最小化窗口 (与 Metro 应用无关)
  14. Windows 键 + D：显示桌面，第二次键击恢复桌面 (不恢复 Metro 应用)
  15. Windows 键 + E：打开资源管理器
  16. Windows 键 + L：锁住电脑或切换用户
  17. Windows 键 + R：打开 “运行” 对话框
  18. Windows 键 + 减号：缩小 (放大镜)
  19. Windows 键 + 加号：放大 (放大镜)
  20. Windows 键 + Esc：关闭放大镜

# 常用基本组合键

  1. Ctrl+A：选择所有
  2. Ctrl+C：复制
  3. Ctrl+E：选择搜索框（资源管理器）
  4. Ctrl+N：新窗口（资源管理器）
  5. Ctrl+R：刷新
  6. Ctrl+V：粘贴
  7. Ctrl+W：关闭当前窗口（资源管理器）
  8. Ctrl+X：剪切
  9. Ctrl+Y：重做
  10. Ctrl+Z：撤销
  11. Ctrl+Tab：在 Metro 应用历史中循环切换
  12. Alt+Spacebar：快捷菜单
  13. Alt+Tab：应用程序间切换
  14. Ctrl+Shift+N：新建文件夹（资源管理器）
  15. Ctrl+Shift+Esc：任务管理器
  16. Ctrl+Shift + 上下左右箭头：选择一段文字

# 文件夹 / 文件的目录树查看

首先，利用 cmd 进入命令提示符窗口，然后利用 cd 打开到相应的文件夹路径，文件名输入可以用 Tab 键补全；当然，如果你觉得这样一步步打开到相应的文件夹里比较麻烦，那你可以尝试第二种方法，在当前文件夹里，按住 Shift 键然后鼠标右击，会出现一个在此处打开命令窗口的操作，点击就会创建一个命令窗口，并且是指向当前文件夹路径的；紧接着，输入 tree 回车，就会显示在当前文件夹下的子文件夹目录，并以树状形式列出；想要显示出所有文件名的目录（包括子文件夹下的文件）可以在后面加上 "/f"。
如果想要输出到文本上，可以使用 tree /f > tree.txt 命令，就可以把生成的文件目录树形结构写入到当前目录的 tree.txt 文件中了（txt 的命名可以随意）。

# 右 Alt 键组合失效

可能是使用了英式键盘输入，切换成美式英语即可，参考：https://www.dell.com/support/kbdoc/zh-cn/000136390 / 按右 - alt - 键单击不起作用
