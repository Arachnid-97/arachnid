Version：v2rayA 2.2.6.7

# 介绍

v2raya 是一个使用 Web UI 来配置 v2ray 的服务，方便好用，特别适合没有图形界面的 linux server 系统。

主要功能：

  • 自动定时从机场订阅地址中获取更新配置
  • 提供透明代理，系统代理，代理端口
  • 提供路由服务，自动更新 GFWList
  • 可视化的配置页面
  • 几乎所有机场都能支持，完美替代 clash 功能
  • 支持多节点负载均衡

项目地址：https://github.com/v2rayA/v2rayA
